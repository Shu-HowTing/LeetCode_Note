### 串和串匹配

#### 串

**字符串**

一般的，由n个字符构成的串记作：

$S = "a_0 a_1 ··· a_{n-1}"$ , 其中 $a_i$属于$\sum$, $0<=i<n$。$\sum$称之为字符表

**子串**

​                                  $ S.substr(i ,k) = S[i, i+k)$

**前缀**

​                                   $ prefix(S, k) = S.substr(0, k) = s[0,k]$

**后缀**

​                                   $suffix(S, k) = S.substr(n-k, k) = S[n-k, n)$

> 空串是任何字符串的子串，也是任何字符串的的前缀和后缀。

KMP算法

def KMP(text, pattern):
    n = len(text)
    m = len(pattern)
    Next = gen_N(pattern)
    i, j = 0, 0
    while j < m and i < n:
        if text[i] == pattern[j] or j < 0:
            j += 1
            i += 1
        else:
            j = Next[j]
    print((i, j))
    return i - j

# 构造next向量，子串的最长公共前后缀
def gen_N(pattern):
    m = len(pattern)
    Next = [-1] * m
    j = 0
    t = Next[0]
    while j < m - 1:
        if t < 0 or pattern[j] == pattern[t]:
            t += 1
            j += 1
            Next[j] = t
        else:
            t = Next[t]
    return Next
